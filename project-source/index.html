基于 STM32F103 的心电 (ECG) 检测系统

该项目是一个完整的心电检测系统，底层基于 STM32F103 微控制器开发。系统中成功移植了 FreeRTOS 实时操作系统以保证多任务数据采集与处理的实时性，并集成了 FATFS 文件系统用于心电数据的本地 SD 卡存储与管理。

C语言 STM32 标准库 FreeRTOS FATFS

网页自动化监测脚本 (Ticket Monitor)

基于 Selenium 开发的自动化辅助工具。支持直接接管已打开的 Edge 浏览器会话，等待手动登录后按回车键启动。脚本会自动高频检索页面中的“剩余舱位”关键字，并在匹配成功后自动暂停并触发提醒。
