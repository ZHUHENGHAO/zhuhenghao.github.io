1. 遇到什么问题？

在编写网页自动化监测脚本的间隙，我尝试用 Python 的 pyserial 库接收硬件传来的高频数据，发现偶尔会出现丢包现象。

2. 解决方案

后来发现是波特率匹配和缓冲区读取策略的问题。调整了 timeout 参数后完美解决。

import serial
# 测试代码片段
ser = serial.Serial('COM3', 115200, timeout=0.1)
